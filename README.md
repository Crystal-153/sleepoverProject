# sleepoverProject
<h1>Fever Dream</h1>
<h2>This is a visual novel game created using RenPy. Loosely based on Alice in Wonderland, this Choose-Your-Own-Adventure game follows our main character as she dreams of an escaping bunny and makes decisions related to it. Follow along with the bold Queen of Hearts, eccentric Mad Hatter, and even the sly Cheshire Cat in this dreamland. Will you survive?</h2>
<h3>Runs on Windows 10, Linux, and macOS.
https://crystal-153.itch.io/fever-dream is the link to the game, Click Start to begin and load to open any previous sessions. Run through the game and try to get all endings!
</h3>
